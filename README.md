# 🌍 AI Language Translator

A modern AI-powered language translator built using **Python**, **Streamlit**, **Deep Translator**, and **Google Text-to-Speech (gTTS)**.

---

## 📌 Features

- 🌍 Translate text into multiple languages
- 🔄 Swap source and target languages
- 🔊 Listen to translated text (Text-to-Speech)
- 📥 Download translated text as `.txt`
- 🕘 Translation history
- ⚠️ Error handling
- 🎨 Modern and responsive interface

---

## 🛠 Technologies Used

- Python
- Streamlit
- Deep Translator
- Google Text-To-Speech (gTTS)

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_LanguageTranslator.git
```

Go to the project folder

```bash
cd CodeAlpha_LanguageTranslator
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
